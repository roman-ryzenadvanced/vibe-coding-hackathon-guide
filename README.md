## What Is This?

A beautifully branded **Hackathon Ideas Guide** for the Vibe Coding Builders Hackathon, featuring:

- **10 project ideas** leveraging the [last30days-skill](https://github.com/roman-ryzenadvanced/last30days-skill) research engine
- **Step-by-step instructions** for using the skill on chat.z.ai
- **Z.ai + rommark.dev dual branding** with dark mode design
- Professional PDF guide with cover page

## Quick Start

### View the Guide

1. Open `hackathon-ideas-guide.pdf` for the full branded guide
2. Or open `guide.html` in a browser for the interactive web version

### Use the Skill

```bash
# Install the last30days skill
npx skills add roman-ryzenadvanced/last30days-skill -g

# Research any topic on chat.z.ai
/last30days vibe coding tools
/last30days Cursor vs Windsurf vs Bolt
/last30days "AI code editors" --competitors
```
